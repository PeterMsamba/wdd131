Site Purpose

**KayiraMovies** is a subscription-based movie and TV show streaming service. The site provides a user-friendly interface for browsing, searching, and watching a wide library of content, with personalized recommendations and the ability to manage a watchlist and profile.

Scenarios

These questions represent the needs of the target audience (movie/TV enthusiasts looking for entertainment):

  • "I just finished watching a movie and want to find something similarâ€”does the site offer **personalized recommendations** based on my viewing history?"
  • "I'm looking for the **latest documentary** released in the past month. How can I quickly find new arrivals in a specific genre?"
  • "I remember a movie with a specific actor, but I forgot the title. Can I **search by cast member**?"
  • "I need to pause my subscription for the summer. Where can I find and **manage my billing and account settings**?"

Color Schema

The schema uses a dark, sophisticated palette common in entertainment apps, designed to make the movie posters and titles pop.

Color Name Hex Code Usage
**Primary Background** (Dark Gray/Black) #121212 Used for the **main page background** (body, sections). Creates a cinematic and distraction-free viewing experience.
**Accent/Primary Text** (Vibrant Red) #E50914 Used for **call-to-action buttons** (e.g., "Sign Up," "Play"), **primary headings** (`h1`, `h2`), and interactive **accents** (e.g., play icon).
**Secondary Text/Subtle Background** (Light Gray) #A0A0A0 Used for **paragraph text**, subtle background elements (e.g., footer), and inactive navigation links.

Typography

A combination of a classic, bold display font for titles and a highly readable sans-serif for body content.

Font Name Type Usage
**Playfair Display** (Serif) Headings Font Used for the **Site Name**, **Section Headings** (`h1`, `h2`), and important movie/show **titles** to give a premium, dramatic feel.
**Roboto** (Sans-serif) Body/Interface Font Used for all **paragraph text**, **navigation links**, search bars, and detailed information (e.g., cast lists, descriptions) for high **readability** across all devices.
